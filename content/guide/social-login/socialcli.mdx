### Add/Update Login Method 

With [Loginradius CLI](https://github.com/LoginRadius/lr-cli/), You can add social provider using the simple command below

   ```sh
   $ lr add social
   ? Select the provider from the list: google
   ? Client ID: ###Client ID###
   ? Client Secret: ###Client Secret###
   google is added as your login Method
   ```

With [Loginradius CLI](https://github.com/LoginRadius/lr-cli/), You can update social provider using the simple command below

   ```sh
   $ lr set social -p google
   ? API Key: ###key###
   ? API Secret: ###secret###
   google updated successfully.
   ```

### Step 3: Whitelist Domain

With the help of **<a href="https://github.com/LoginRadius/lr-cli/" target="_blank">LoginRadius CLI</a>** you can whitelist your domain using your terminal.

For security reasons, LoginRadius processes the API calls that are received from the whitelisted domains. Local domains (http://localhost and http://127.0.0.1) are whitelisted by default.

    ```$ lr add domain -d htts://google.com
    htts://google.com is now added as your whitelisted domain.
    ```
